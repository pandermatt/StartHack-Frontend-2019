## Get Started

1. Clone the project

```
git clone https://github.com/react-native-training/react-native-elements-app.git
```

2. Install dependencies

```
cd react-native-elements-app

npm install
```

3. Run the app locally

```
npm start - this should pop up Expo Dev Tools.
```
